# EstatisticaParaCienciaDeDados

Conceitos de estatística
	POPULAÇÃO é o principal alvo de interesse para satisfazer um objetivo. Caso a população seja grande demais
	deve-se selecionar uma AMOSTRA, sendo um subconjunto da POPULAÇÃO. 
	Quando os dados(percentuais, médias, medianas etc) são apurados sobre a POPULAÇÃO chamam-se PARÂMETROS.
	Já, quando são apurados sobre AMOSTRAS, são conhecidos como ESTATÍSTICAS.
	
	DADOS ou VARIÁVEIS são informações coletadas em uma pesquisa. Dependendo da natureza dessa informação são
	classificados em:
		QUALITATIVAS (CATEGÓRICAS) categorias, qualidades atributos, características
			NOMINAIS nomes, rótulos, categorias que não tem ordem ou hierarquia
			ORDINAIS relação de ordem do menor para o maior
		QUANTITATIVAS (NUMÉRICAS) medidas, contagens, numéricas
			DISCRETAS resultam de um conjunto finito ou enumerável de valores possíveis (contagens)
			CONTÍNUAS resultam em um número infinito de valores possíveis (mensuráveis)
