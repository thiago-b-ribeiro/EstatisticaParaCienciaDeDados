# EstatisticaParaCienciaDeDados

## Conceitos de estatística

- **POPULAÇÃO**: o principal alvo de interesse para satisfazer um objetivo
  - **PARÂMETROS**: dados apurados sobre a população (percentuais, médias, medianas, etc)
- **AMOSTRA**: subconjunto da população utilizado quando a população é muito grande
  - **ESTATÍSTICAS**: dados apurados sobre amostras
- **DADOS/VARIÁVEIS**: informações coletadas em uma pesquisa, classificados em:
  - **Qualitativas (Categóricas)**: Categorias, qualidades, atributos, características
    - **Nominais**: nomes, rótulos, categorias que não têm ordem ou hierarquia
    - **Ordinais**: relação de ordem do menor para o maior
  - **Quantitativas (Numéricas)**: Medidas, contagens, informações numéricas
    - **Discretas**: resultam de um conjunto finito ou enumerável de valores possíveis (contagens)
    - **Contínuas**: resultam em um número infinito de valores possíveis (mensuráveis)

## Medidas de Tendência Central
- **MÉDIA** 
  - A média da **população** é calculada como $$\mu = \frac{1}{N} \sum_{i=1}^{N} x_i$$
  - Já a média da **amostra** é calculada como $$\bar{x} = \frac{\sum x_i}{n}$$
- **MEDIANA**: valor que divide a amostra em duas partes iguais. Para localicar a mediana é necessário que a amostra esteja ordenada e que sejam calculadas $$Posicao1 = \frac{N}{2}$$ $$Posicao2 = \frac{N + 2}{2}$$ e por fim $$\bar{x} = \frac{ValorPosicao1 + ValorPosicao2}{2}$$
- **MODA**: valor que aparece com maior frequência.
